# Zapret-GUI
Версия Zapret, но с улучшением в виде полноценного интерфейса и возможностью не держать открытой консоль (Приложение сворачивается в трей). #ZapretGUI

Зачем вам нужен графический интерфейс?
Как минимум для удобства использования. Приятно видеть не просто окно консоли, а иконку в трее или графический интерфейс.

!Программа адаптирована под любые версии Zapret просто нужно указать путь к папке, листы остановка перезапуск всё есть внутри интерфейса.

Возможно дизайн вам покажетс немного странный и скучный, но пока так, возможно, буду обновлять. (В планах добавить сразу ещё и обход телеграмма)
